Step 2 / 7
Node.js の確認
「Node.js」とは、Claude Code を動かすエンジンです。すでに入っているか確認しましょう。
1. 黒い画面（ターミナル）を開いてください
2. 下のコマンドをコピーして、黒い画面に貼り付けてEnterを押してください
node --version
✅ Node.js が確認できました！「次へ」を押してください。
Node.js がインストールされていません。
下のボタンからインストールしてください。
Node.js の公式サイトを開く →

「LTS版」と書かれているものをダウンロードして、インストールが終わったら
もう一度 node --version を実行してください。

「command not found」「'node'は認識されない」などが表示された場合は未インストールです。

Step 3 / 7
Claude Code をインストール
コマンド1つでインストールできます。黒い画面に文字が大量に流れますが、正常です。
ターミナルで実行してください（数分かかります）
npm install -g @anthropic-ai/claude-code
よくある質問： 「黒い画面に文字がたくさん流れるけど大丈夫？」
→ 大丈夫です。インストール中の正常な動きです。止まるまで待ちましょう。
EACCES エラーが出た場合（Mac）：
sudo npm install -g @anthropic-ai/claude-code
パスワードを聞かれたら、PCのログインパスワードを入力してください（入力中は何も表示されません。正常です）。
network error が出た場合： Wi-Fiの接続を確認して、もう一度試してください。 会社のネットワークの場合はプロキシ設定が必要なことがあります。
Step 4 / 7
Anthropic アカウントと API キーの取得
Claude Code を使うには「API キー」が必要です。無料で取得できます（使用量に応じて課金されます）。
手順：
  1. 下のボタンから Anthropic Console を開く
  2. アカウントを作成する（メールアドレスで登録）
  3. 「API Keys」から「Create Key」をクリック
  4. キーをコピーして下に貼り付ける
Anthropic Console を開く →
✅ APIキーを確認しました。「次へ」を押してください。
⚠️ APIキーは sk- で始まる長い文字列です。貼り付け直してください。
💡 APIキーは大切に管理してください
他の人に教えないでください。このウィザードは、キーをどこかに送信することはありません。
Step 5 / 7
API キーを設定する
取得した API キーをPCに覚えさせます。「環境変数」という仕組みを使います。難しくありません。コマンドをコピーして実行するだけです。
環境変数とは、PCが「秘密の情報」を覚えておく仕組みです。
APIキーをコードに直接書くと危険なので、こうして別管理します。
一度設定すると、ターミナルを開くたびに自動で読み込まれます。
どちらを使っているか確認してください：
ターミナルの上部に「bash」か「zsh」と表示されています。 わからなければ zsh でOKです（Mac は標準 zsh）。

シェルの種類：

コマンドプロンプト（PowerShell）で実行してください。
APIキーが自動で埋め込まれています。そのままコピーしてOKです。
PowerShell / コマンドプロンプトで実行
実行後、ターミナル（PowerShell）を一度閉じて、新しく開き直してください。
Step 6 / 7
動作確認
Claude Code が正しくインストールされているか確認します。
ターミナルで実行してください
claude --version

結果はどちらでしたか？

🎉 セットアップ完了まであと一歩！「次へ」を押してください。
よくある原因と対処法：
  • ターミナルを一度閉じて、新しく開いてみてください
  • それでも出る場合は Step 3 に戻ってインストールし直してください
  • Mac で「command not found」の場合：export PATH="$PATH:$(npm prefix -g)/bin" を実行してみてください

❓ よくある質問

Q. ターミナルって何ですか？
コンピューターに文字でコマンドを入力する画面です。
Mac:「ターミナル.app」/ Windows:「Windows PowerShell」または「コマンドプロンプト」を検索してください。
Q. APIキーにお金はかかりますか？
アカウント作成は無料です。使用量に応じて課金されます。最初は少量で試せます。Anthropic Console で使用量を確認できます。
Q. Claude Code と Claude（ブラウザ版）の違いは？
Claude Code はあなたのPC上のファイルを直接操作できます。ブラウザ版は会話のみです。Claude Code はより「仕事をしてくれる」AIです。
Q. 途中で失敗しても大丈夫？
大丈夫です。このウィザードはいつでも「戻る」ボタンで前のステップに戻れます。
Q. インストールに失敗したら？
エラーメッセージをそのまま Claude（ブラウザ版）に貼り付けて「解決策を教えて」と聞くと解決法を教えてくれます。
